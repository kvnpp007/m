Centrar una página

Utilizando la propiedad margin de CSS, es muy sencillo centrar una página web horizontalmente. La solución consiste en agrupar todos los elementos de página en un elemento div y asignarle a ese div margenes laterales automáticos. El div que encierra los contenidos suele llamar contenedor (en ingles se denomina grapper o container).

Como se sabe, el valor 0 auto significa que los margenes superiores son iguales a 0 y los margenes laterales toman un valor de auto, esto hara qué los navegadores centren ese elemento respecto a la ventana del navegador.
